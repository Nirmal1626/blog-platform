# BlogVerse — Modern Blogging Platform

A full-stack blogging platform built with **Next.js** and **Supabase**, featuring AI-powered summaries via the **Google Gemini API**.

![Next.js](https://img.shields.io/badge/Next.js-15-black?logo=next.js)
![Supabase](https://img.shields.io/badge/Supabase-Database%20%26%20Auth-3FCF8E?logo=supabase)
![Google AI](https://img.shields.io/badge/Google%20Gemini-AI%20Summaries-4285F4?logo=google)

## Features

- **Authentication** — Sign up / sign in with email & password via Supabase Auth
- **Role-Based Access** — Author, Viewer, and Admin roles with granular permissions
- **Blog Posts** — Create, edit, delete posts with featured images
- **AI Summaries** — Automatic ~200-word summaries generated by Google Gemini API
- **Comments** — Logged-in users can comment on posts; admins can moderate
- **Search** — Filter posts by title with debounced search
- **Pagination** — Server-side paginated post listing
- **Premium Dark UI** — Glassmorphism design with smooth animations

## User Roles

| Role   | Permissions                                       |
| ------ | ------------------------------------------------- |
| Author | Create posts · Edit own posts · View comments     |
| Viewer | View posts · Read summaries · Comment on posts    |
| Admin  | View all posts · Edit any post · Monitor comments |

## Project Setup

### Prerequisites

- Node.js 18+
- npm
- A [Supabase](https://supabase.com) project
- A [Google AI API key](https://aistudio.google.com/apikey)

### 1. Clone the Repository

```bash
git clone <your-repo-url>
cd blog-platform
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env.local` file in the project root:

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url_here
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key_here
GOOGLE_AI_API_KEY=your_google_ai_api_key_here
```

### 4. Set Up the Database

1. Go to your Supabase project dashboard
2. Navigate to **SQL Editor**
3. Copy and run the contents of `supabase/schema.sql`
4. This creates the `users`, `posts`, and `comments` tables with Row Level Security policies

### 5. Configure Supabase Auth

1. In Supabase dashboard, go to **Authentication** > **Providers**
2. Ensure **Email** provider is enabled
3. (Optional) Disable email confirmation for development: **Authentication** > **Settings** > **Email Auth** > Toggle off "Confirm email"

### 6. Run Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Go to [vercel.com](https://vercel.com) and import your repository
3. Add environment variables in Vercel dashboard:
   - `NEXT_PUBLIC_SUPABASE_URL`
   - `NEXT_PUBLIC_SUPABASE_ANON_KEY`
   - `GOOGLE_AI_API_KEY`
4. Deploy!

### Deploy to Netlify

1. Push your code to GitHub
2. Go to [netlify.com](https://netlify.com) and import your repository
3. Set build command: `npm run build`
4. Set publish directory: `.next`
5. Add environment variables in Netlify dashboard
6. Install the Next.js plugin for Netlify
7. Deploy!

## AI Feature

When a new post is created, the platform automatically:

1. **Generates a ~200-word summary** using the Google Gemini API
2. **Stores the summary** in the database alongside the post
3. **Displays the summary** on the post listing page and post detail page

The summary can also be regenerated from the edit post page.
